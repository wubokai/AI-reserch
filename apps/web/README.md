# Web

AI Quant Research Assistant 的 Next.js 前端骨架。

## 本地命令

- `pnpm dev`：开发服务器。
- `pnpm lint`：ESLint。
- `pnpm typecheck`：TypeScript strict 检查。
- `pnpm test`：Vitest 与 Testing Library。
- `pnpm build`：Next.js standalone 构建。
- `pnpm e2e`：Playwright Chromium smoke test。
- `pnpm check`：依次执行 lint、typecheck、test、build。

当前页面只展示显式标注的 Mock/DEMO 研究工作流，不包含真实金融数字。
